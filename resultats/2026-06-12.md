- **Budget** : 880–2 990 €/mois (max tolérance ~3 300 €)
- **Type** : Location — loft/atelier/surface, maison de caractère, duplex atypique

- **Localisations** : Pas de contrainte fixe — cherche sur toute la France (signal faible). Zones observées dans les annonces aimées : Tourcoing (59), Paris 17e (75), Brains-sur-les-Marches (53).

  1. **Esprit loft / grands volumes / double hauteur** — présent dans les 3 annonces aimées, et très visible sur les photos
  2. **Espace extérieur privatif** (terrasse bois, jardin) — toutes les annonces aimées en ont
  3. **Mélange ancien + industriel** (poutres, cheminée, parquet, pierre apparente) — récurrent notes + photos
  4. **Lumière / grandes baies vitrées** — mentionné explicitement et visible
  5. **Espace bureau / télétravail** — explicite dans les notes
  6. **Espace pour travail du corps / mouvement** — explicite dans les notes (terrasse, atelier)
  7. **Pas de vis-à-vis intrusifs** — explicite dans les notes
  8. **Parquet / sol de qualité**
  9. **Façade propre, sans fioritures**
  10. **Cuisine et salle de bain propres** (pas "hideuses")
